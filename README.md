---

## ⚙️ Функционал проекта

### 1. Обработка формы (`process.php`)
Данные из `form.html` передаются методом **POST**. Сервер выполняет следующие действия:
- Очистка данных через `htmlspecialchars()`.
- Валидация: проверка на заполнение обязательных полей (ФИО и Дата).
- Сохранение ошибок в сессию при неудачной проверке.

### 2. Работа с сессиями и файлами
- **Sessions:** После успешной отправки данные текущего пользователя доступны на главной странице до закрытия браузера.
- **Files (`data.txt`):** Все регистрации записываются в файл в формате CSV (разделитель `;`), что позволяет хранить историю всех клиентов.

### 3. Просмотр данных (`view.php`)
Создана отдельная страница для чтения файла `data.txt`. Данные парсятся с помощью функции `explode()` и выводятся в виде структурированного списка.

## 🚨 Выполнение штрафного задания
Реализовано хранение данных в **Cookies**:
- Имя последнего зарегистрированного пользователя сохраняется в браузере на 1 час.
- При возвращении на `index.php` (даже после закрытия браузера) система приветствует пользователя по имени, считанному из Cookie.
