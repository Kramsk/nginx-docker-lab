## ⚙️ Ход выполнения работы

### Шаг 1. Конфигурация Docker-окружения
В файл `docker-compose.yml` был добавлен сервис PHP и настроена зависимость для Nginx:

```
services:
  php:
    image: php:8.2-fpm
    container_name: lab2_php
    volumes:
      - ./www:/var/www/html
    expose:
      - "9000"

  nginx:
    # ... предыдущие настройки
    depends_on:
      - php
```

### Шаг 2. Настройка Nginx для обработки PHP

В конфигурационный файл `nginx/default.conf` добавлена поддержка FastCGI:

```
location ~ \.php$ {
    include fastcgi_params;
    fastcgi_pass php:9000;
    fastcgi_index index.php;
    fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
}
```

### Шаг 3. Проверка работоспособности PHP

Создан файл `www/index.php`. При переходе по адресу `http://localhost:8080` успешно отображается системная информация PHP.

Согласно варианту «Регистрация в спортзале», форма включает:

    ФИО (текстовое поле)

    Дата рождения (поле даты)

#### Обработка на JavaScript:

  Использование addEventListener("submit", ...) для перехвата события отправки.

  Метод e.preventDefault() для предотвращения перезагрузки страницы.

  Динамический вывод данных в блок <div id="result"> с применением стилизованной рамки.
